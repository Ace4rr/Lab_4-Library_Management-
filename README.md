# Лабораторная работа

## Описание

Данный проект представляет собой симуляцию с пользовательскими коллекциями и псевдослучайной моделью поведения библиотеки, реализованную в рамках **Лабораторной работы №4**

### Поддерживаемые возможности

* **Псевдолучайная генерация событий в библиотеке, зависящая от seed**

### Типы книг(два производных класса)
 - Rarebook
 - Textbook
